# Python 3
En caso de no tener Python 3:
- Windows
  Descargar instalador desde la [página oficial](https://www.python.org/downloads/) e instalar.
- Linux
  ~~~
  sudo apt-get install python3
  ~~~

# Librerías necesarias
Numpy y regex.
Para instalar:
~~~
pip3 install numpy, regex
~~~
